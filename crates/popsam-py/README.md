# popsam

Python bindings for `popsam`, a tool for AI-assisted selection of semantically representative texts from larger text collections.

The package exposes the same core functionality as the Rust library:

- selection from precomputed embeddings
- local embedding generation with Candle
- embedding generation through an OpenAI-compatible API

See the repository README for the full project overview and examples:

https://github.com/curoli/popsam
